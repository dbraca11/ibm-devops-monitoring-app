# ibm-devops-monitoring-app
Proyecto de automatización de monitoreo y despliegue de microservicios (Certificación IBM DevOps).

# Monitoreo TI & DevOps Project (IBM Certification)

## Descripción
Este proyecto es una implementación práctica de los conocimientos adquiridos en el **IBM DevOps and Software Engineering Professional Certificate**. 
Combina mi experiencia real en el monitoreo de sistemas críticos en **Bancaribe** con herramientas modernas de ingeniería.

## Tecnologías utilizadas
* **Lenguaje:** Python (Automatización)
* **Contenedores:** Docker (Estandarización)
* **Metodología:** Agile / DevOps

## Cómo ejecutarlo
1. Construir la imagen: `docker build -t monitoring-app .`
2. Correr el contenedor: `docker run monitoring-app`
